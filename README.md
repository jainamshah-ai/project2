# Pattern Generator and Number Analyzer

## Description
This is a simple Python console-based project that allows users to:

- Generate star (`*`) patterns
- Analyze numbers within a range
- Identify even and odd numbers
- Calculate the sum of numbers in a range

The program uses a menu-driven interface and continues running until the user chooses to exit.

---

## Features

### 1. Pattern Generator
Creates a triangle star pattern based on the number of rows entered by the user.

Example:
*
**
***
****
*****

---

### 2. Number Analyzer
Analyzes numbers in a given range and:
- Checks whether each number is even or odd
- Displays the result
- Calculates the total sum of the range

Example:
the number 1 is odd
the number 2 is even
the number 3 is odd

Sum = 6

---

### 3. Exit Option
Allows the user to safely close the program.
